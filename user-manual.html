Guide to the Simulator



  1. Start by first entering your Choice, either enter the values for each Road yourself keeping in mind the limits or let the default values be filled.
  2. For user input, enter the values in text field and correspondingly space is calculated.
  3. These values have to be entered for each road of the Round-about
  4. While for Default input, only time has to be entered for which roads have to be filled with the data.
  5. The Facts & Observations provide you the insights of the project.
  6. The Records tell you the previous data for the roads and their corresponding Congestion.
  7. While, the Report clearly explains the project idea and it background working.
  8. Everything in the project is worked on units(u), where each unit is equal to 2.5 meters.
  9. The space for the roads and vehicles is also calculated in terms of units(u).
  10. After the values are entered, the output for each road is displayed which explains the throughput for the roads.
  11. Warnings are also given when the values entered exceed the maximum limit.
